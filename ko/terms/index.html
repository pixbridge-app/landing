이 문서는 PixBridge 초기 테스트 이용에 관한 일반적인 규칙을 설명하는 초안입니다. 정식 출시 전 운영 주체가 검토하고 수정할 수 있습니다.

PixBridge로 돌아가기 English

1. 개요

이 이용약관은 PixBridge와 PixBridge 모바일 앱, 랜딩 페이지, 정책 페이지, 예제 저장소, 초기 테스트 자료를 이용하거나 테스트하는 사용자 사이의 일반적인 이용 조건을 설명하는 초안입니다.

사용자는 PixBridge가 초기 테스트 단계에 있으며, 기능, 가용성, 데이터 처리 방식, 가격 정책, 약관이 변경될 수 있음을 이해해야 합니다.

2. 서비스 설명

PixBridge는 모바일 Camera-to-API 클라이언트입니다. 사용자는 서버 주소를 등록하고, 요청 방식 또는 전송 형식을 설정하며, 필요한 경우 인증 토큰을 입력하고, 모바일 카메라로 이미지를 촬영해 사용자가 설정한 서버 API로 전송할 수 있습니다. 이후 서버 API가 반환한 JSON 응답에서 사용자가 설정한 필드를 앱 화면에 표시할 수 있습니다.

일반적인 앱 사용 과정에서 PixBridge 자체 서버가 이미지를 분석하는 서비스를 제공하지 않습니다. 연결된 API의 분석, 저장, 처리, 응답 방식은 사용자가 선택한 서버에 의해 결정됩니다.

3. 초기 테스트 안내

PixBridge는 현재 초기 테스트 또는 테스트 준비 단계입니다. 서비스에는 오류, 미완성 기능, 변경 중인 인터페이스, 제한된 가용성, 테스트 전용 동작이 포함될 수 있습니다.

사용자는 PixBridge를 중요 업무, 긴급 상황, 정식 운영, 법적으로 민감한 워크플로에 사용하기 전에 위험을 독립적으로 검토하고 수용해야 합니다.

4. 사용자 책임

  • 사용자는 서버 URL, API 설정, 요청 설정, 토큰 관리, PixBridge를 통해 업로드하는 이미지에 대한 책임을 집니다.
  • 사용자는 이미지를 촬영, 업로드, 전송, 처리할 권리와 권한을 가지고 있어야 합니다.
  • 사용자는 개인정보, 민감정보, 기밀정보 또는 제3자 정보가 포함된 이미지를 전송할 때 적절한 권한과 보호 조치를 확인해야 합니다.
  • 사용자는 인증 토큰을 보호하고, 과도한 권한이 있는 토큰을 피하며, 더 이상 필요하지 않은 토큰을 폐기해야 합니다.
  • 사용자는 자신의 이용에 적용되는 법률, 계약, 제3자 API 약관을 준수할 책임이 있습니다.

5. 사용자 설정 서버 및 API

PixBridge는 사용자가 설정한 API 엔드포인트로 이미지를 전송합니다. 해당 서버가 이미지를 저장, 분석, 공유, 삭제, 보호하는 방식은 PixBridge가 통제하지 않습니다.

사용자는 본인이 소유, 관리, 사용 권한을 보유하거나 충분히 검토한 서버와 API만 PixBridge에 연결해야 합니다.

6. 금지 행위

사용자는 PixBridge를 다음 목적으로 사용해서는 안 됩니다.

  • 관련 법령 또는 타인의 권리를 침해하는 행위
  • 무단 감시 또는 모니터링
  • 적절한 권리, 동의, 권한 없이 이미지를 업로드하는 행위
  • 제3자 API를 남용, 과부하, 공격하거나 제한을 우회하려는 행위
  • PixBridge, 관련 인프라, 다른 사용자에게 장애를 일으키려는 행위
  • 악성 파일, 유해 콘텐츠, 권한 없는 인증 정보를 제출하는 행위

7. 제3자 서비스

PixBridge는 Firebase 또는 유사한 백엔드 서비스, Tally 또는 유사한 폼 서비스, GitHub Pages, 분석 또는 로그 도구, 앱 스토어, 사용자가 설정한 API와 같은 제3자 서비스에 의존할 수 있습니다.

제3자 서비스는 각자의 약관, 정책, 보안 관행, 가용성 제한을 가질 수 있습니다. PixBridge는 타인이 통제하는 제3자 시스템에 대해 책임을 지지 않을 수 있습니다.

8. 전문적 조언 아님

PixBridge는 Camera-to-API 워크플로를 위한 소프트웨어 도구입니다. 법률, 의료, 금융, 보안, 컴플라이언스 또는 기타 전문적 조언을 제공하지 않습니다. 필요한 경우 사용자는 전문가의 검토를 받아야 합니다.

9. 서비스 변경 및 가용성

PixBridge는 초기 테스트 중 기능을 추가, 제거, 중단, 변경할 수 있습니다. 접근은 제한, 중단, 종료될 수 있으며 문서, 예시, 정책, 인터페이스는 시간이 지나며 업데이트될 수 있습니다.

10. 보증 부인

PixBridge는 초기 테스트 단계에서 "있는 그대로" 및 "이용 가능한 상태"로 제공됩니다. 관련 법률이 허용하는 범위에서 PixBridge는 중단 없는 가용성, 오류 없는 작동, 특정 목적 적합성, 모든 서버 API와의 호환성을 보장하지 않습니다.

11. 책임 제한

관련 법률이 허용하는 범위에서 PixBridge는 간접적, 부수적, 특별, 결과적, 징벌적 손해 또는 사용자가 설정한 서버, 제3자 API, 사용자가 제공한 토큰, 업로드 이미지, PixBridge가 통제하지 않는 외부 서비스로 인한 손실에 대해 책임을 지지 않을 수 있습니다.

일부 관할권에서는 특정 책임 제한이 허용되지 않을 수 있으므로, 이 조항의 일부는 모든 사용자에게 동일하게 적용되지 않을 수 있습니다.

12. 이용 종료

사용자가 이 약관을 위반하거나, 보안 위험을 만들거나, 서비스를 남용하거나, 타인에게 피해를 줄 수 있는 방식으로 PixBridge를 사용하는 경우 PixBridge는 테스트 기능 접근을 중단하거나 제한할 수 있습니다. 사용자는 언제든지 PixBridge 이용을 중단할 수 있습니다.

13. 약관 변경

PixBridge는 앱, 테스트 프로그램, 제3자 서비스, 비즈니스 모델 변경에 따라 이 이용약관을 업데이트할 수 있습니다. 중요한 변경이 있는 경우 시행일을 갱신합니다.
